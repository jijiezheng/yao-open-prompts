# Prompts

这里按场景分类存放拆分后的提示词文件。每个文件都包含统一 frontmatter、简短说明和可复制的 Prompt 正文。

分类目录：

- `01-ai-methods`：元提示词、反编译、提示词工程方法
- `02-ai-work`：企业调研、客服、销售、助教、PPT、网页等工作流
- `03-ai-learning`：学习方法、记忆术、习惯养成
- `04-ai-life`：健康、亲子歌曲等生活场景
- `05-ai-education`：儿童教育和游戏创作
- `06-ai-content`：写作、标题、视频、图像和 PPT 创意
- `07-ai-coding`：架构与编程协作
- `08-ai-marketing`：GEO 内容生成与改写
- `09-ai-thinking`：思维、记忆和标题灵感类提示词
